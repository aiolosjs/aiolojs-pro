## 基于antd pro 改版的中台系统脚手架

``` bash
npm install
```

or

``` bash
yarn
```

### 启动

``` bash
npm start
```

### 打包

``` bash
npm run build
```

### Check code style

``` bash
npm run lint
```

You can also use script to auto fix some lint error:

``` bash
npm run lint:fix
```

### Test code

``` bash
npm test
```
